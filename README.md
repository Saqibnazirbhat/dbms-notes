# DBMS Notes

Free, interactive study notes for Database Management Systems, written for
beginners and organized as four weekly chapters. Every hard idea comes with an
interactive figure you can play with: step through a query the way the engine
runs it, crash a transaction, test keys on live data, and more.

**📖 Read online:** https://Saqibnazirbhat.github.io/dbms-notes/

**⬇ Study offline:** download [`index.html`](index.html) and open it in any
browser. Everything works without internet.

## What's covered

- **Week 1 · Course Overview** — what a database is, why file systems fail,
  levels of abstraction, schemas, DDL vs DML, inside the database engine
- **Week 2 · Relational Model and Basic SQL** — relations, keys,
  SELECT–FROM–WHERE, LIKE patterns, NULL and three-valued logic,
  GROUP BY / HAVING, set operations, the Cartesian product
- **Week 3 · Intermediate and Advanced SQL** — subqueries, SOME/ALL, EXISTS,
  joins, views, transactions and ACID, foreign-key actions, triggers
- **Week 4 · Relational Query Languages and Database Design** — relational
  algebra, division, relational calculus, ER diagrams, cardinality,
  weak entities, mapping diagrams to tables

## How to use them

Read the chapters in order; each builds on the last. Don't skip the figures —
press their Step and Run buttons and drag the sliders, because watching an idea
move is how it sticks. Answer each **Check your understanding** box before
opening the answer, and before a test, revisit the **When things go wrong**
section at the end of each chapter.
